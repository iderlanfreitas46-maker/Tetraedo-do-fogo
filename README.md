# Tetraedo-do-fogo
O fogo necessita da presença simultânea de quatro elementos para existir e se manter: calor, combustível, oxigênio e a reação em cadeia que perpetua o processo. A remoção de qualquer um deles interrompe a combustão — esse é o princípio por trás de todos os métodos de extinção de incêndio (resfriamento, abafamento, retirada do material ).
